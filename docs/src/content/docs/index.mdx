# ryu_ldn_nx

**ryu_ldn_nx** is a Nintendo Switch sysmodule that enables online local multiplayer (LDN) gaming by connecting to Ryujinx LDN servers over TCP. Play "local wireless" games like Mario Kart 8 with friends anywhere in the world!

## Features

- **Easy Installation** — Just extract to your SD card and reboot
- **Transparent Operation** — Games work without modification
- **PIA Mesh Compatibility** — Broadcast delivery ensures PIA mesh discovery works correctly
- **Tesla Overlay** — Monitor connection status and configure settings in-game
- **Auto Reconnect** — Fast first retry (200ms) then exponential backoff with jitter
- **Low Latency** — Optimized for real-time gameplay with optional P2P mode

## Supported Games

Works with games that use Nintendo's LDN service:

- Mario Kart 8 Deluxe
- Super Smash Bros. Ultimate
- Monster Hunter Rise
- Animal Crossing: New Horizons
- And many more...

## Quick Start

```bash
# 1. Download the latest release from GitHub
```
